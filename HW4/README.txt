Install python 3.6
libraries re and time should be standard libraries

Run: python search.py or python3 search.py
Also can run in IDE like SublimeText.

Program will print out information for each search for all three intranets{1,5,7}
----

In the scripts, you may change the start node.

you may also pass an integer into Best_FS to indicate the queue size for beam search.
Not all beam widths are guranteed to find a solution if it exists.
